# Taofik.com

A responsive personal portfolio website built using HTML, CSS, and JavaScript. This project showcases my profile, projects, skills, and contact information through a clean and organized multi-page layout.

## About

This repository contains the source code for **Taofik.com**, a personal portfolio website designed to present my work and provide visitors with an easy way to learn more about me.

## Features

- Fully Responsive Design
- Multi-Page Layout
- About Page
- Projects Page
- Contact Page
- Smooth Navigation
- Organized File Structure
- Fast Loading
- Mobile Friendly

## Live Demo

Open the website locally or deploy it using GitHub Pages or Vercel.

## Installation

Clone this repository:

```bash
git clone https://github.com/sadmanmahmudtaofiq/Taofik.com.git
```

Navigate to the project folder:

```bash
cd Taofik.com
```

Open the `HTML/index.html` file in your browser or launch it with **VS Code Live Server**.
